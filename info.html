Projeto de Final de Modulo 1 - T14

A Proposta é criar um jogo de texto com 3 personagens e 3 fases para cada um deles. Usando a interação via prompt.

Primeiro você se identifica como maior de idade para poder jogar. Após a validação deste dado, você pode selecionar um personagem.

O primeiro passo na jornada é treinar seu personagem para a luta classificatoria. Cada modalidade de treino aumenta a força do personagem.

Após o treino você está pronto para a luta. Encare um advesario e escolha suas melhores habilidades para vencer. Marcando 3 pontos você se classifica e recebendo -3 você é derrotado.

Se você ganhou a luta, está na hora de enfrentar o campeão, desta vez você deve marcar 7 pontos, mas se receber -3 será derrotado.

Receba o cinturão, comemore e retorne para jogar novamente.
